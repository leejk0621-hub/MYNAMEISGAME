# GAME_RULE.md

## 1. 게임 개요 (Game Overview)
*   **게임 이름**: Fruit Catcher
*   **장르**: 동작 인식 아케이드 게임
*   **목표**: 하늘에서 떨어지는 과일을 바구니로 받아 점수를 획득하고, 폭탄을 피하세요.
*   **플레이 방식**: 웹캠 앞에서 몸을 움직여(Left, Center, Right) 바구니를 이동시킵니다.

## 2. 조작 방법 (Controls)
화면은 3개의 레인(Lane)으로 구분됩니다. 플레이어의 포즈에 따라 바구니가 해당 레인으로 즉시 이동합니다.

| 포즈 (Pose) | 인식 내용 | 바구니 위치 |
| :--- | :--- | :--- |
| **Left** | 왼쪽 포즈 (고개 왼쪽 / 왼손 들기) | 왼쪽 레인 (Lane 1) |
| **Center** | 정면 / 차려 자세 | 중앙 레인 (Lane 2) |
| **Right** | 오른쪽 포즈 (고개 오른쪽 / 오른손 들기) | 오른쪽 레인 (Lane 3) |

> **참고**: 모델의 라벨은 `Left`, `Center`, `Right` 입니다. (코드에서 대소문자 처리 예정)

## 3. 아이템 및 점수 (Items & Scoring)
아이템은 랜덤한 레인에서 생성되어 아래로 떨어집니다.

| 아이템 | 이미지 | 효과 | 점수 | 확률 |
| :--- | :--- | :--- | :--- | :--- |
| **사과 (Apple)** | 🍎 | 기본 득점 | +100점 | 30% |
| **포도 (Grape)** | 🍇 | 추가 득점 | +150점 | 20% |
| **바나나 (Banana)** | 🍌 | 보너스 득점 | +200점 | 15% |
| **오렌지 (Orange)** | 🍊 | 고득점 | +300점 | 15% |
| **황금사과 (Gold)** | 🌟 | 대박 찬스 | +500점 | 5% |
| **폭탄 (Bomb)** | 💣 | 라이프 감소 | **Life -1** | 15% |

## 4. 게임 규칙 (Rules)
1.  **시작 & 종료**
    *   `Start` 버튼 클릭 시 게임 시작 (카운트다운 3초).
    *   라이프가 0이 되면 게임 종료 (Game Over).
2.  **레벨 시스템**
    *   초기 속도: 느림
    *   시간이 지날수록(혹은 득점 할수록) 아이템 낙하 속도가 빨라짐.
3.  **판정**
    *   바구니와 아이템이 겹치면 획득/피격 판정.
    *   아이템은 획득/피격 후 사라짐.
    *   바닥에 닿은 아이템은 사라짐 (Miss 페널티 없음).

## 5. UI 구성
*   **HUD**: 점수(Score), 라이프(❤️ ❤️ ❤️)
*   **Main**: 웹캠 화면 위 오버레이 (바구니, 떨어지는 아이템)
*   **End**: 최종 점수 표시 + '다시하기' 버튼

## 6. 개발 단계
1.  **Step 1**: `gameEngine.js` 구조 변경 (Fruit Catcher 전용 로직)
2.  **Step 2**: 3개 레인 및 아이템 낙하 구현
3.  **Step 3**: 포즈 인식(`poseEngine.js`)과 바구니 이동 연동
4.  **Step 4**: 충돌 처리 및 점수/라이프 로직
5.  **Step 5**: UI 및 게임 오버 처리
